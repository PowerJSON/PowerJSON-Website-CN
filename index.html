PowerJSON

—— 由 JSON 改进而来的数据格式



支持 注释、多行字符串、导入文件、运算符 等等由 JSON 扩展而来的功能.

// comment
/*
    comment
 */

{
    text: 'hello world',
    MultiLineText: `
        Welcome
        to
        PowerJSON!
    `,

    // file and import
    myFile: new File('./file.txt'),
    importPJSONFile: new PJSON('./index2.pjson'),
    getFile: new GET('http://www.powerjson.org'),

    // operator
    tenDaySeconds: 60 * 60 * 24 * 10
}
